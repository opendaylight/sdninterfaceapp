SdniNorthbound

This class SDNi REST APIs is used to retrieve SDNi Network Capabilities Data. And also Update the available bandwidth based on the parameters Its queries SDNiAggregator to fetch/modify the data. Totally 3 API calls - Network Capabilities Data, Update

The following resources are applicable:

  • /{containerName}/property/{propertyName}/{propertyValue}
  • /{containerName}/property/linkbandwidth/{propertyValue}
  • /{containerName}/qos
  • /{containerName}/topology

/{containerName}/property/{propertyName}/{propertyValue}

Mount Point: /controller/nb/v2/sdni/{containerName}/property/{propertyName}/{propertyValue}

PUT

Update the available bandwidth property. This method returns a non-successful response if the bandwidth is not updated.

Parameters

name description type default
containerName Name of the Container (Eg. 'default') path
propertyName Name of the Property. Properties that can be configured are: description, forwarding(only for default container) and tier path
propertyValue Value of the Property. Description can be any string (Eg. 'Node1'), valid values for tier are non negative numbers, and valid values for forwarding are 0 for reactive and 1 for proactive forwarding. path

Response Body

element: (custom)
media types: */*
application/xml

Response as dictated by the HTTP Response Status code 

Example:
Request URL:
http://localhost:8080/controller/nb/v2/sdni/default/property/bandwidth/5

Status Codes

HTTP Status Code Description
201 Created Operation successful
400 Bad Request The nodeId or configuration is invalid
401 Unauthorized User not authorized to perform this operation
404 Not Found The Container Name or node or configuration name is not found
406 Not Acceptable The property cannot be configured in non-default container
409 Conflict Unable to update configuration due to cluster conflict or conflicting description property
503 Service Unavailable One or more of Controller services are unavailable
